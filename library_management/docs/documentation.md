# Library Management System (Go)

## Overview
A console-based system to manage library operations such as adding, removing, borrowing, and returning books.

### Key Features
- Add and remove books
- Borrow and return books
- Track available and borrowed books
- Manage members

### Technologies Used
- **Language:** Go (1.24)
- **Architecture:** Layered (MVC-like)
  - Models: Data structures
  - Services: Business logic
  - Controllers: CLI interface

### How to Run
```bash
go mod tidy
go run main.go
